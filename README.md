# PhD Chapter 3 – Environmental Drivers of Atlantic Salmon Survival
This repository contains the full data processing and analysis workflow used in Chapter 3 of my PhD research. The objective of this chapter is to quantify how environmental conditions across freshwater and marine ecosystems influence Atlantic salmon survival in two index rivers in Quebec, Canada.

## Project Structure
- `R/01_prepare_data` – Extraction and preparation of environmental and biological datasets  
- `R/02_clean_data` – Cleaning, merging, and covariate construction  
- `R/03_analysis` – Statistical modelling and analyses  

## Data
Raw datasets are not stored in this repository due to size and access restrictions. Processed data used for analyses are (or will be) available in the `data/processed` folder when permitted.

## Reproducibility
Analyses were conducted in R.
